# Nuestro Álbum Secreto ❤️

Una galería de fotos privada y mágica para parejas de larga distancia, construida con amor.

## ✨ Características
* **Visualizador Polaroid:** Las fotos se muestran con un hermoso estilo de tarjeta tipo Polaroid.
* **Túnel Mágico en la Nube:** Si uno de los dos sube, elimina o crea una carpeta de fotos, la otra computadora lo sincroniza automáticamente vía un "Buzón Secreto" (conectado a Firebase Realtime Database).
* **Almacenamiento Local Seguro:** Las fotos residen exclusivamente en la computadora. El buzón borrará cualquier rastro después de la sincronización.
* **100% Personalizado:** Desarrollado sobre Electron y Node.js para sentirse como una aplicación de escritorio nativa, rápida y ligera (con mucho rojo, flores animadas y letras cursivas).

## 🚀 Cómo Empezar

Esta aplicación es portátil y funciona creando un identificador único la primera vez que se abre. 

1. Ve a la parte de la derecha de esta página que dice **"Releases"**.
2. Descarga el archivo **`Nuestro Album Setup 1.0.0.exe`**.
3. Haz doble clic y se abrirá automáticamente la galería.
